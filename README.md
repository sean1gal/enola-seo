## Setup

**Requirements:** Claude Code with browser access.

**Install:**
1. Copy the `enola-seo/` folder into your Claude Code project directory
2. Open Claude Code in that directory
3. Say: `run my SEO audit` or `/enola`

**Second session and beyond:**
Enola reads your saved context automatically. No re-interviewing.
Just say what you want to run.

---

## Slash Commands

Fast one-shot tasks — no interview required:

```
/enola audit [url]    Full technical scan
/enola geo [url]      AI visibility check
/enola fix [url]      Hard blockers only — what's broken now
/enola score [url]    Content quality + EEAT score
/enola report         Export audit to PDF
```

## The 26 Audits

5 categories: Google Business Profile (01–08), Website (09–13), Authority (14–16), Content (17–20), Advanced (21–26).

Advanced audits cover: Technical SEO, GEO (AI search visibility), Content Quality EEAT scoring, Hard Blockers, Content Refresh, and PDF Export.

Full audit menu with descriptions is in `SKILL.md`.

## File Output

Everything saves to `/seo-audit/` in your working directory — one `.md` file per audit, plus `BUSINESS_CONTEXT.md`, `DIAGNOSIS.md`, `CLAUDE.md`, `ALERT-LOG.md`, and `12-WEEK-PLAN.md` on wrap-up. Full file list in `SKILL.md`.
