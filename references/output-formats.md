# ENOLA SEO — OUTPUT FORMATS

Every audit file saved to `/seo-audit/[##]-[slug].md`

Standard header — every file:

```
[AUDIT NAME] — [Business Name]
Date: [date]
Model: [B2C Local / B2B Multi]
Impact: HIGH / MED / LOW
Timeline: [X days / weeks to see results]
───────────────────────────────────────
```

---

## FORMAT BY AUDIT

**01 — GBP Categories**
Table: Category | Tier | Searches Unlocked | Add This Week
Numbered add-list below, highest priority first
How-to instructions per category

**02 — Attributes**
Table: Attribute | Impact | CTR Effect | Tier | How to Add
This-week list at bottom (Tier 1 only)

**03 — Review Teardown**
Table: Competitor | Total | Last 30d | Last 60d | Last 90d | Top Keywords
Velocity gap + my monthly target
Top 10 phrases to train customers to use
Review request script (copy-paste block)

**04 — Review Responses**
Labeled blocks: [5A] [5B] [5C] [4A] [4B] [4C] [3A] [3B] [3C] [1A] [1B] [1C]
All 12 written in full
Usage note per tier

**05 — Posts Strategy**
Table: Week | Day | Type | Full Copy | Image Description
Weeks 1-4: full copy
Weeks 5-8: topic + type outline

**06 — Services Copy**
Per service block:

```
SERVICE: [name]
OPTIMIZED TITLE: [title]
DESCRIPTION: [40-60 words]
KEYWORD HIT: [keyword]
AREA MENTION: [city/area]
OUTCOME STATED: [what customer gets]
```

Missing services flagged at bottom

**07 — GBP Description**

```
VERSION 1 — KEYWORD FOCUS
[copy]
Chars: X/750 | Keywords: [list] | Areas: [list]

VERSION 2 — CONVERSION FOCUS
[copy]
Chars: X/750 | Urgency tactic: [what]

VERSION 3 — TRUST FOCUS
[copy]
Chars: X/750 | Trust signals: [what]

TEST RECOMMENDATION: Start with V[X] — measure [metric] over 30 days
```

**08 — Photo Plan**
Table: Week | Shot Type | What | Where | Why It Ranks
Naming convention examples (5+ real examples)
Geotagging instructions block

**09 — Keyword Gaps**
Table: Keyword | Volume | Difficulty | Intent Stage | Page Needed | Action
Sorted highest priority first
Top 5 immediate actions boxed at bottom

**10 — Money Pages**
Per page block:

```
PAGE: [url]
PROBLEM: [specific issue]
NEW TITLE TAG: [exact text]
NEW H1: [exact text]
NEW META DESCRIPTION: [exact text]
CONTENT TO ADD: [what + ~word count]
INTERNAL LINKS TO ADD: [from page → anchor text]
IMPACT: HIGH/MED/LOW
```

30-day sprint calendar below

**11 — City Pages**
Full page written per city, clearly separated
Headers: ## [City] — [Service] Page
All sections complete, paste-ready

**12 — Search Console Sprint**
Same format as Audit 10 per page
30-day sprint table at bottom
Quick wins (under 10 min) flagged separately

**13 — Review Sentiment**
Section 1: Language Audit
  - Emotional words list (20)
  - Specific outcomes list (10)
  - Pre-service fears list (5)
  - Money phrases list
  - 5-star vs 3-star language differences
Section 2: Rewritten Copy
  - GBP description rewrite
  - Homepage headline + subheadline
  - Review request script
  - 3 social proof statements

**14 — Backlinks**
Table: Month | Link | Target | Tactic | Contact
All outreach emails written in full below table
Labeled: [Email 1-A], [Email 1-B], etc.

**15 — Citations**
Table: Platform | Listed | Name ✓ | Address ✓ | Phone ✓ | Issues
Red flags section (FAIL items only)
Missing platforms list
Fix instructions per platform
Monthly maintenance checklist

**16 — Intent Map**
Table: Stage | Example Keywords (10) | Volume | Content Type | Conv. Rate
Stage 4 priority section below:
Per keyword: current status + 90-day ranking plan

**17 — Content Gaps**
Summary table: Keyword | Category | Volume | Priority
Full brief block per keyword (all 20):

```
KEYWORD: [keyword]
CATEGORY: [problem-awareness / comparison / local service]
TITLE: [SEO title]
URL: /[slug]
TARGET KW: [primary] + [secondary 1], [secondary 2]
QUESTIONS TO ANSWER: [3 bullets]
WORD COUNT: [n]
INTERNAL LINKS: [from → to]
CTA: [what and where]
BRIEF: [200-word summary of what page covers]
```

**18 — Entity**
Section 1: Current entity status (knowledge panel Y/N, Wikidata Y/N)
Section 2: Schema code block (complete JSON-LD)
Section 3: Platform checklist (table: Platform | Priority | Status | Action)
Section 4: 90-day plan (week by week)

**19 — Posting Patterns**
Section 1: Pattern analysis (table: Day | Frequency | Best Types | Seasonal Notes)
Section 2: Strategy recommendation (3-5 sentences, specific to findings)
Section 3: 4-week calendar
  Table: Week | Day | Time | Type | Full Copy | Keyword | CTA | Image

**20 — Monthly Report**
The template itself, formatted and ready to use
Sections with [FILL IN] placeholders only for the variable numbers
5-minute review checklist below template

**21 — Technical Scan**
Scorecard table: Category | Check | Status | Issue | Fix | Impact
Hard Blockers section (boxed, urgent)
Quick Wins section (under 10 min each)
Full fix instructions per FAIL/WARN item

**22 — GEO Audit**
GEO scorecard: Dimension | Score | Status | Gap
AI search status (what's showing, what's not)
FAQ copy (5 questions + answers, written in full)
Citation bait content brief
Schema additions block

**23 — Content Quality Score**
Score table: Page | Exp | Expert | Auth | Trust | Depth | Total | Grade
Thin content flags list
Priority fixes ranked by score gap
Rewrite brief for lowest-scoring page (full brief)
One rewritten intro paragraph per page that scored under 30

**24 — Hard Blockers**
Three sections only:
CRITICAL — fix today (numbered list, exact steps)
IMPORTANT — fix this week (numbered list, exact steps)
CLEAN — no issues (list)
No padding. No explanations beyond what's needed to fix it.

**25 — Content Refresh Plan**
Decay audit table: Page | Age | Issues | Priority | Action
Full refresh brief for top 3 pages
Rewritten intro paragraph for each top 3

**26 — PDF Export**
Confirm: file path + file size + sections included
Note: "Open /seo-audit/26-pdf-export/[filename].pdf"
