# PRESET — LOCAL QUICK START

**Goal:** fastest map pack wins for a local service business
**Time:** one afternoon
**Sequence:** Prompt 24 → 01 → 02 → 22

## Why this order

24 clears any active blockers first — phone not clickable, GBP not verified, broken contact form. These are calls going to competitors today. Fix them before anything else.

01 and 02 are the fastest ranking levers in local SEO. Categories and attributes directly control which searches trigger your listing in the map pack. One correct category can put you in front of searches you're currently missing entirely. Changes can appear in GBP Insights within days.

22 sets your AI search baseline from day one. Google AI Overviews, ChatGPT, Perplexity, and Bing Copilot now answer local service questions directly. Build these signals early — they compound slowly and the window is open now.

## How to run

Load your business context first (or run the diagnostic to set it up), then run each audit in sequence. Use the output of each as input context for the next.
